Ziel klonen
Diese Einstellung bestimmt, wohin Ihre WordPress-Site geklont wird. Zu den Optionen gehört das Klonen innerhalb derselben Domäne, in eine vorhandene oder neue Domäne oder in eine Subdomäne. Geben Sie ein Verzeichnis für den Klon an oder lassen Sie das Feld leer, um direkt in das Stammverzeichnis der ausgewählten Domäne oder Subdomäne zu klonen.
