Pingbacks aktivieren
Wenn Sie Pingbacks aktivieren, erlauben Sie WordPress, Benachrichtigungen an andere Blogs zu senden, wenn Sie in Ihren Inhalten auf diese verlinken und umgekehrt. Wenn Sie beispielsweise einen Beitrag schreiben und einen Link zu einem anderen Blog einfügen, erhält dieser Blog eine Pingback-Benachrichtigung. Sie können diese Benachrichtigung als Kommentar anzeigen und ihren Lesern so zeigen, dass Sie auf ihre Inhalte verlinkt haben.

Auf diese Weise geben Sie die Quelle an und können beim Networking und beim Aufbau von Beziehungen zu anderen Bloggern helfen.
