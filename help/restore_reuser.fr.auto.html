Régénérer le nom d'utilisateur s'il est pris ?
Par défaut, Virtualmin ne vous permettra pas de restaurer un domaine si la connexion d'administration est déjà utilisée sur ce système, par exemple par un autre domaine portant un nom similaire. Cependant, changer cette option sur Oui forcera la réattribution du nom d'utilisateur, ce qui évitera un conflit. Cela peut être utile si vous utilisez example.com et souhaitez restaurer une sauvegarde de example.org à partir d'un autre système.
