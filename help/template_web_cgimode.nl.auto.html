Uitvoeringsmodus van CGI-script
Deze optie bepaalt of en hoe CGI-scripts standaard worden uitgevoerd voor nieuwe virtuele servers. De mogelijke opties zijn:
  • CGI-scripts uitgeschakeld - scripts kunnen helemaal niet worden uitgevoerd.
  • FCGIwrap - scripts worden uitgevoerd via een FCGIwrap-server op de achtergrond, één per domein.
  • suEXEC - scripts worden gestart via de Apache suexec- wrapper.
