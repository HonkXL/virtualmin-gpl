Speicherlimit
Dies ist die maximale Speichermenge, die ein Skript auf Ihrer WordPress-Site zuweisen darf. Es ist, als würden Sie ein Limit festlegen, wie viel Speicher einzelne Aufgaben gleichzeitig verwenden können. Wenn Ihre Site klein bis mittelgroß ist, reichen möglicherweise die standardmäßigen 40 MB aus. Wenn Sie jedoch viele Plugins oder komplexe Skripte haben, müssen Sie dieses Limit möglicherweise erhöhen.
