Activer le mode maintenance
Cette option vous permet de mettre votre site dans un état où les visiteurs savent que vous effectuez des mises à jour ou des modifications.

Lorsque le mode de maintenance est activé, les visiteurs verront un message indiquant que votre site est en mode de maintenance. Vous pouvez personnaliser la page de maintenance en plaçant un fichier maintenance.php personnalisé dans le répertoire wp-content .
