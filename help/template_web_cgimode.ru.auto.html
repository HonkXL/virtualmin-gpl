Режим выполнения CGI-скрипта
Этот параметр определяет, будут ли и как сценарии CGI выполняться по умолчанию для новых виртуальных серверов. Возможные варианты:
  • CGI-скрипты отключены — скрипты вообще не могут быть запущены.
  • FCGIwrap — скрипты запускаются через фоновый сервер FCGIwrap, по одному на домен.
  • suEXEC — скрипты запускаются через обертку Apache suexec .
