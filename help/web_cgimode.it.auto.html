Questa opzione determina come vengono eseguiti gli script CGI per questo server virtuale. Le opzioni possibili sono:
  • Script CGI disabilitati : gli script non possono essere eseguiti affatto.
  • Server FCGIwrap : gli script vengono eseguiti tramite un server in background, uno per dominio.
  • Wrapper suEXEC : gli script vengono avviati tramite il wrapper suexec Apache. Tieni presente che questa opzione potrebbe non essere accessibile su tutti i sistemi.
