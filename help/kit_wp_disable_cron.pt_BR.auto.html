Desabilitar agendador de tarefas cron
Desativa o agendamento automático de tarefas cron do WordPress, o que é útil durante migrações de banco de dados ou ao usar um agendador externo para evitar conflitos.

No entanto, geralmente não é recomendado desabilitar o agendador de tarefas cron, pois ele é usado pelo WordPress para executar várias tarefas, como verificar atualizações, publicar postagens agendadas e muito mais.
