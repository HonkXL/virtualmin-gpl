Modo de ejecución de script CGI
Esta opción determina si los scripts CGI se ejecutan de forma predeterminada y cómo se ejecutan para los nuevos servidores virtuales. Las opciones posibles son:
  • Scripts CGI deshabilitados : los scripts no se pueden ejecutar en absoluto.
  • FCGIwrap : los scripts se ejecutan a través de un servidor FCGIwrap en segundo plano, uno por dominio.
  • suEXEC : los scripts se inician a través del contenedor Apache suexec .
