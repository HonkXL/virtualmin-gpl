Description du site
Il s'agit généralement d'un court texte qui explique le sujet de votre site. Il est souvent affiché juste sous le nom du site sur votre page d'accueil et peut également être utilisé dans les métadonnées que les moteurs de recherche utilisent pour décrire votre site dans les résultats de recherche. Cette description peut être importante pour le référencement, car elle peut influencer la décision des utilisateurs de cliquer sur votre site à partir des résultats de recherche.
