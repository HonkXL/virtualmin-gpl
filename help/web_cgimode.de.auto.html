Ausführungsmodus für CGI-Skripte
Diese Option bestimmt, wie CGI-Skripte für diese virtuellen Server ausgeführt werden. Die möglichen Optionen sind:
  • CGI-Skripte deaktiviert – Skripte können überhaupt nicht ausgeführt werden.
  • FCGIwrap-Server – Skripte werden über einen Hintergrundserver ausgeführt, eines pro Domäne.
  • suEXEC-Wrapper – Skripte werden über den Apache- Suexec- Wrapper gestartet. Beachten Sie, dass diese Option möglicherweise nicht auf allen Systemen verfügbar ist.
