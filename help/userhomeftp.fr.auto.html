Répertoire d'accueil
Par défaut, le répertoire de base d'un utilisateur FTP de site Web est le même que le répertoire Web de base du domaine. Cependant, vous pouvez le modifier en un sous-répertoire pour limiter l'accès de l'utilisateur à ce seul répertoire. Cela suppose que le serveur FTP est configuré pour restreindre les clients à leurs répertoires personnels.
