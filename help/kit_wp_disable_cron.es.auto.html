Deshabilitar el programador de trabajos cron
Desactiva la programación automática de trabajos cron de WordPress, lo cual es útil durante las migraciones de bases de datos o cuando se utiliza un programador externo para evitar conflictos.

Sin embargo, generalmente no se recomienda deshabilitar el programador de trabajos cron, ya que WordPress lo utiliza para realizar diversas tareas, como buscar actualizaciones, publicar publicaciones programadas y más.
