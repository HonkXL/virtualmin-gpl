Создать домен хоста по умолчанию с сертификатом Let's Encrypt?
Если для этого параметра установлено значение Да , Virtualmin автоматически попытается создать скрытый домен хоста по умолчанию (виртуальный сервер) с сертификатом Let's Encrypt во время следующей проверки конфигурации. Если сертификат Let's Encrypt не может быть успешно запрошен для имени хоста системы, домен хоста по умолчанию будет автоматически удален.

Однако можно использовать параметр «Да» и «Сохранить видимость», чтобы сохранить домен хоста по умолчанию видимым и не удалять его в случае, если сертификат Let's Encrypt не может быть успешно запрошен, что позволит в дальнейшем внести коррективы вручную.

Если имя хоста системы изменится, Virtualmin автоматически попытается соответствующим образом обновить доменное имя хоста по умолчанию при следующей проверке конфигурации.

Домен хоста по умолчанию можно отключить, установив для этого параметра значение Нет, что приведет к его удалению из системы при следующей проверке конфигурации.

Имейте в виду, что использование домена хоста по умолчанию для почты крайне не рекомендуется и может вызвать проблемы с доставкой почты!
