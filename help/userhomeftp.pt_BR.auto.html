Diretório inicial
Por padrão, o diretório home para um usuário FTP de site é o mesmo que o diretório web base do domínio. No entanto, você pode alterá-lo para um subdiretório para limitar o acesso do usuário somente a esse diretório. Isso pressupõe que o servidor FTP esteja configurado para restringir clientes a seus diretórios home.
