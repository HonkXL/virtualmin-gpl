¿Crear un dominio predeterminado de host con el certificado Let's Encrypt?
Cuando esta opción está configurada en Sí , Virtualmin intentará automáticamente crear un dominio predeterminado de host oculto (servidor virtual) con un certificado Let's Encrypt durante la siguiente verificación de configuración. Si el certificado Let's Encrypt no se puede solicitar correctamente para el nombre de host del sistema, el dominio predeterminado del host se eliminará automáticamente.

Sin embargo, es posible utilizar la opción Sí y mantener visible para mantener visible el dominio predeterminado del host y no eliminarlo en caso de que el certificado Let's Encrypt no se pueda solicitar correctamente, lo que permitirá realizar ajustes manuales más adelante.

Si el nombre de host del sistema cambia, Virtualmin intentará automáticamente actualizar el nombre de dominio predeterminado del host en consecuencia en la siguiente verificación de configuración.

El dominio predeterminado del host se puede desactivar configurando esta opción en No, lo que hará que se elimine del sistema en la siguiente verificación de configuración.

Tenga en cuenta que no se recomienda utilizar el dominio predeterminado del host para el correo, ya que puede causar problemas con la entrega del correo.
