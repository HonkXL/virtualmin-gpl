Creare un dominio predefinito dell'host con il certificato Let's Encrypt?
Quando questa opzione è impostata su Sì , Virtualmin tenterà automaticamente di creare un dominio predefinito host nascosto (server virtuale) con un certificato Let's Encrypt durante il successivo controllo di configurazione. Se non è possibile richiedere correttamente il certificato Let's Encrypt per il nome host del sistema, il dominio predefinito dell'host verrà automaticamente rimosso.

Tuttavia, è possibile utilizzare l'opzione Sì e mantieni visibile per mantenere visibile il dominio predefinito dell'host e non rimuoverlo nel caso in cui il certificato Let's Encrypt non possa essere richiesto con successo, il che consentirà modifiche manuali in seguito.

Se il nome host del sistema cambia, Virtualmin tenterà automaticamente di aggiornare di conseguenza il nome di dominio predefinito dell'host al successivo controllo di configurazione.

Il dominio predefinito dell'host può essere disabilitato impostando questa opzione su No, che lo farà rimuovere dal sistema al successivo controllo della configurazione.

Tieni presente che l'utilizzo del dominio predefinito dell'host per la posta è altamente sconsigliato e potrebbe causare problemi con la consegna della posta!
