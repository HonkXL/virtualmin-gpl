Ausführungsmodus für CGI-Skripte
Diese Option bestimmt, ob und wie CGI-Skripte standardmäßig für neue virtuelle Server ausgeführt werden. Die möglichen Optionen sind:
  • CGI-Skripte deaktiviert – Skripte können überhaupt nicht ausgeführt werden.
  • FCGIwrap – Skripte werden über einen Hintergrund-FCGIwrap-Server ausgeführt, eines pro Domäne.
  • suEXEC – Skripte werden über den Apache suexec- Wrapper gestartet.
