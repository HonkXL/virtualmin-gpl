使用 Let's Encrypt 證書創建主機默認域？
當此選項設置為Yes時，Virtualmin 將在下一次配置檢查期間自動嘗試使用 Let's Encrypt 證書創建隱藏主機默認域（虛擬服務器）。如果系統主機名無法成功請求 Let's Encrypt 證書，主機默認域將被自動刪除。

但是，可以使用“是”並保持可見選項來保持主機默認域可見，並且在無法成功請求 Let's Encrypt 證書的情況下不將其刪除，這將允許稍後進行手動調整。

如果系統的主機名發生更改，Virtualmin 將在下一次配置檢查時自動嘗試相應地更新主機默認域名。

可以通過將此選項設置為“否”來禁用主機默認域，這將使其在下次配置檢查時從系統中刪除。

請注意，強烈建議不要使用主機默認域來發送郵件，這可能會導致郵件傳送出現問題！
