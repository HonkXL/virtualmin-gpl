Home-Verzeichnis
In diesem Feld können Sie ein benutzerdefiniertes Home-Verzeichnis für einen Postfachbenutzer auswählen. Sie möchten beispielsweise, dass sich das Home-Verzeichnis im Verzeichnis ` public_html befindet, damit der Benutzer problemlos Webdateien erstellen kann. In den meisten Fällen sollte die Einstellung jedoch auf >Automatisch eingestellt werden.
