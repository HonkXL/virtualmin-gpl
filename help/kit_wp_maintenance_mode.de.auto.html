Wartungsmodus aktivieren
Mit dieser Option können Sie Ihre Site in einen Zustand versetzen, in dem Besucher wissen, dass Sie Aktualisierungen oder Änderungen vornehmen.

Wenn der Wartungsmodus aktiviert ist, wird Besuchern eine Meldung angezeigt, dass sich Ihre Site im Wartungsmodus befindet. Sie können die Wartungsseite anpassen, indem Sie die benutzerdefinierte Datei maintenance.php im Verzeichnis wp-content ablegen.
