Limite de memória
Esta é a quantidade máxima de memória que um script tem permissão para alocar no seu site WordPress. É como definir um limite de quanta memória tarefas individuais podem usar por vez. Se o seu site for pequeno ou médio, o padrão de 40M pode ser o suficiente. Mas se você tiver muitos plugins ou scripts complexos, pode ser necessário aumentar esse limite.
