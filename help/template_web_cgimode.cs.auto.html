Režim provádění skriptu CGI
Tato možnost určuje, zda a jak jsou skripty CGI standardně spouštěny pro nové virtuální servery. Možné možnosti jsou:
  • CGI skripty vypnuty - skripty nelze vůbec spouštět.
  • FCGIwrap - skripty jsou spouštěny přes server FCGIwrap na pozadí, jeden na doménu.
  • suEXEC - skripty se spouštějí přes Apache suexec wrapper.
