CGI-skriptutførelsesmodus
Dette alternativet bestemmer om og hvordan CGI-skript kjøres som standard for nye virtuelle servere. De mulige alternativene er:
  • CGI-skript deaktivert - skript kan ikke kjøres i det hele tatt.
  • FCGIwrap - skript kjøres via en FCGIwrap-server i bakgrunnen, ett per domene.
  • suEXEC - skript lanseres via Apache suexec -innpakningen.
