Limite de mémoire
Il s'agit de la quantité maximale de mémoire qu'un script est autorisé à allouer sur votre site WordPress. C'est comme fixer une limite sur la quantité de mémoire que les tâches individuelles peuvent utiliser à la fois. Si votre site est de petite ou moyenne taille, la valeur par défaut de 40 Mo peut suffire. Mais si vous avez beaucoup de plugins ou de scripts complexes, vous devrez peut-être augmenter cette limite.
