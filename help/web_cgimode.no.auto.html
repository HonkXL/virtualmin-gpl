CGI-skriptutførelsesmodus
Dette alternativet bestemmer hvordan CGI-skript utføres for denne virtuelle serveren. De mulige alternativene er:
  • CGI-skript deaktivert - skript kan ikke kjøres i det hele tatt.
  • FCGIwrap-server – skript kjøres via en bakgrunnsserver, ett per domene.
  • suEXEC wrapper - skript lanseres via Apache suexec wrapper. Merk at dette alternativet kanskje ikke er tilgjengelig på alle systemer.
