使用 Let's Encrypt 证书创建主机默认域？
当此选项设置为Yes时，Virtualmin 将在下一次配置检查期间自动尝试使用 Let's Encrypt 证书创建隐藏主机默认域（虚拟服务器）。如果系统主机名无法成功请求 Let's Encrypt 证书，主机默认域将被自动删除。

但是，可以使用“是”并保持可见选项来保持主机默认域可见，并且在无法成功请求 Let's Encrypt 证书的情况下不将其删除，这将允许稍后进行手动调整。

如果系统的主机名发生更改，Virtualmin 将在下一次配置检查时自动尝试相应地更新主机默认域名。

可以通过将此选项设置为“否”来禁用主机默认域，这将使其在下次配置检查时从系统中删除。

请注意，强烈建议不要使用主机默认域来发送邮件，这可能会导致邮件传送出现问题！
